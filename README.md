#WeatherView

WeatherView is an Android Library that helps you make a cool weather animation for your app.


##Usage
###Basic usage
Add this to your **build.gradle**:
``` 
repositories {
        maven { url = 'https://jitpack.io' }
    }
dependencies {
    compile 'com.github.MatteoBattilana:WeatherView:1.0'
}

```
